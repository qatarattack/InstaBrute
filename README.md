Cracking instagram account password by giving passwordlist file and support proxy 

###Usage

```./instabrute.py <username> <wordlist> <proxylist> <thread>```

``` ./instabrute.py netcat words.txt proxy.txt 4```

###Requirements

* requests
* multiprocessing
